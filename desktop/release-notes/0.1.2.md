## What’s new in My AI Library 0.1.2

- Added the Windows Electron desktop application.
- Added a branded, assisted NSIS installer with desktop and Start menu shortcuts.
- Bundled the local FastAPI backend and supervised its startup and shutdown.
- Moved writable application data to `%LOCALAPPDATA%\MyAILibrary` so upgrades do not replace user data.
- Added native file and folder selection while keeping the renderer sandboxed.
- Added local FFmpeg support and packaged frontend assets for offline startup.
