## What’s new in My AI Library 0.1.4 Beta 3

- Added an opt-in Testing update channel for unsigned GitHub prereleases.
- Added update availability notifications, download progress, restart prompts, and a post-update success message.
- Added versioned Markdown release notes that users can read before downloading and after updating.
- Kept the Stable channel restricted to signed production releases.
- Added clear security messaging before users enable unsigned updates.
- Preserved verified pre-update backups, database integrity checks, migration recovery, and user data across updates.
- Pinned the bundled Windows FFmpeg download to an immutable, SHA-256-verified build for reproducible installers.
- Fixed clean GitHub runner support for packaging the Python backend.
